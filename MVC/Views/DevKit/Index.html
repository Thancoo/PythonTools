Sha512 Hash
动态调用：{{'{{sha512("param")}}'}}

MD5 Hash

动态调用：{{'{{md5("param")}}'}}

Unix Timestamp

动态调用：{{'{{timeStamp()}}'}}

Url Encode

动态调用：
{{'编码：{{urlEncode("param")}}'}}
{{'解码：{{urlDecode("param")}}'}}

Dynamic String Generate

动态调用：
这是一个例子
{{'http://xxxxx?UnixTime={{timeStamp}}&Token={{sha512(account+password)}}'}}
timeStamp为系统提供，如要的到正确的结果，请提供account和password参数！
